## 🎯 Project Overview

NayePankh Foundation is a non-profit organization empowering underprivileged youth through education, skill development, and volunteer-driven initiatives. This analytics dashboard was built during a **Data Analytics Internship** to provide the foundation with a centralized, visual, and actionable view of their key performance metrics.

The dashboard covers **8 core areas** of the organization — from fundraising and volunteer engagement to social media growth, campaign ROI, and website performance — with data-driven recommendations for each.

> ⚠️ All figures are **representative/simulated** for internship demonstration purposes.

## 🚀 Run Locally

### Prerequisites
- **Node.js 16+** — [Download here](https://nodejs.org)
- **Git** — [Download here](https://git-scm.com)

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/haseeb-28/NayePankh-analytics-dashboard.git

# 2. Enter project folder
cd NayePankh-analytics-dashboard

# 3. Install dependencies
npm install --legacy-peer-deps

# 4. Start development server
npm run dev

# 5. Open in browser
# → http://localhost:5173
```

### Build for Production

```bash
npm run build
npm run preview
```

## 📈 Key Insights from the Data

- 💸 **Donor Retention Risk** — 44% of donors are one-time givers. A nurture email campaign could add **₨3.2L/year**
- 📱 **Instagram ROI** — Grew **272% YTD** with 5.9% engagement rate — highest of all platforms
- 🌙 **Ramadan Campaign** — Delivered **3.8× ROI**, highest of all campaigns. Double budget recommended for 2025
- 🌐 **Website Bounce Rate** — Homepage at 72% bounce. Above-the-fold CTA fix could reduce it by **25%**
- 👥 **Volunteer Churn** — Early warning emails for 21+ day inactive volunteers can recover **30–40%** before they leave
